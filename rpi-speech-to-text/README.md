Mini speech to text program that I will be deploying on Raspberry Pi. 


Hardware:
* Raspberry Pi 4 Model B
* ReSpeaker Mic Array v2.0
* MicroSD (>32GB)

Peripheral & Connectors (optional)
* USB Keyboard
* USB Mouse
* Monitor
* HDMI + Micro HDMI Adaptor

Software
* Raspberry Pi Imager
* Raspberry Pi OS (64-bit)
